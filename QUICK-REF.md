## 🎮 How to Use

### Open a Chat
```
Click any contact in left sidebar
```

### Send a Message
```
1. Type message in input box
2. Press Enter or click send (✈️)
3. Auto-reply comes in ~1 second
```

### Add Emoji
```
1. Click emoji button (😊)
2. Click any emoji
3. Emoji added to message
```

### Search Chats
```
Type in search box at top
Chat list filters in real-time
```

### Unread Messages
```
Green badge shows number of unread
Disappears when chat is opened
```

## 🔧 Customization Quick Edits

### Change Primary Color
```css

.icon-btn:hover { color: #YOUR_COLOR; }
```

### Add New Contact
```javascript
/* In script.js, add to mockChats array */
{
    id: 6,
    name: 'Your Name',
    lastMessage: 'Hi!',
    timestamp: 'Now',
    unread: 0,
    online: true,
    messages: [
        { type: 'received', text: 'Hello!', time: 'Now' }
    ]
}
```
